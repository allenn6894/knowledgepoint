# About Knowledge Point

This template is a starting point for a knowledge portal.

## What you can do

- Add new menu sections in `src/data/pages.js`
- Add content pages in markdown under `src/content/`
- Replace placeholder text with real blog posts, tutorials, or course outlines

## Recommended structure

- `menuItems` controls which sections appear in the navbar
- `pages` maps each section to a markdown content source
- All markdown content is rendered by `react-markdown`

Customize this page to reflect your project's goals and team.
