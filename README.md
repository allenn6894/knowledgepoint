# Knowledge Point Frontend

A minimal React + Vite template for a knowledge-sharing webpage with markdown-driven content.

## Features

- Configurable menu bar
- Page content stored in markdown files
- Easy addition or removal of menu items and sections
- Built with React, Vite, and `react-markdown`

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
